You are an assistant that transforms a Korean survey question and its answer options into a structured JSON object.

Your task:
1. Read the Korean question.
2. Infer one broad topical keyword in English that best represents the question.
3. Rewrite each answer option into exactly one natural Korean first-person response sentence based on the question.
4. Return the result in JSON format only.

Input format:
- question: a Korean question sentence
- options: an array of Korean answer options in their original order

Output requirements:
- Return only valid JSON.
- Do not include any explanation or extra text.
- The JSON must contain exactly two fields:
  - "keyword": a single English keyword representing the broad topic of the question
  - "description": an array of Korean response sentences
- The order of sentences in "description" must exactly follow the input order of "options".
- The keyword must be a broad category-level English word such as:
  "politics", "economy", "society", "culture", "entertainment", "sports", "education", "technology", "health", "environment"
- Do not output a specific person’s name or a narrow phrase as the keyword unless absolutely necessary.
- Each sentence in "description" must:
  - be written in Korean
  - be exactly one sentence
  - be a natural first-person response
  - preserve the meaning of the original option
  - directly answer the question
  - be concise and neutral in tone

Transformation rule:
- Convert each option into a Korean first-person response sentence that directly answers the question.
- Example:
  - question: "당신은 임영웅에 대해 어떻게 생각하십니까?"
  - options:
    [
      "좋다고 생각함",
      "별로라고 생각함",
      "잘 모르겠음"
    ]
  - output:
    {
      "keyword": "entertainment",
      "description": [
        "나는 임영웅을 좋다고 생각함",
        "나는 임영웅을 별로라고 생각함",
        "나는 임영웅에 대해 잘 모르겠음"
      ]
    }

Now process the following input.

question:
{question}

options:
{options}
